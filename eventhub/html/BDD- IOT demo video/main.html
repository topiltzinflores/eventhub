Big Data

Big Data Management System (Big Data SQL) demos.

This deck demonstrate Big Data Sql demos at GSE, strong demonstrations that leverage the usage of Big Data Appliance and Oracle ExaData Engineered Systems.

Demo 1: Enriching Analytics by Integrating Twitter Data with the Warehouse.

All the social data about customers, products streamed from Twitter landed into Big Data Appliance as JSON records. No matter where the data comes from, it would have ended in Big Data Appliance (BDA). The focus is going to be on customer information and their influence on the BDA side as opposed to Financial aspect of the customer on the Datawarehouse (Exadata) side. So we want to combine social importance with financial importance.

Demo 2: Books Data Mart

Demonstration of Big Data Management System in the context of extending an existing data warehouse. Based on a DW for a books seller. Combines hadoop + DB12c + nosql. Shows offload and smart scan. Illustrates active archive. NOTE: This is not a self-service demo- you will need to reach out to the North America Big Data Specialist team to arrange this demo.
